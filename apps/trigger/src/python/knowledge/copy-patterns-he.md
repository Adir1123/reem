# Copy Patterns — Hebrew (Personal Finance Tips)

The few-shot library that Pass B reads alongside `brand-voice-he.md` (the
philosophy) and the Hebrew exemplar images (the visual target). Every line
below represents native-quality finance Hebrew. When in doubt, imitate
these patterns.

---

## Cross-cutting rules

### English density

- **Headlines**: high English tolerance — brand/account names welcome
  (`Roth IRA לפנסיה`, `Vanguard vs Fidelity`).
- **Bodies**: sparse English — only unavoidable specific terms (`ETF`,
  `Vanguard`, `Roth`, `401k`). 2+ English content words in a one-line
  body = re-write to flow as Hebrew.

### Address (gender + number)

| Slide role | Default address |
|---|---|
| Hook (slide 1) | Singular masculine OR impersonal |
| Tip slides (2–6) | Singular masculine imperative ("תפתח", "תחסוך") OR passive ("מגדירים", "פותחים") |
| CTA (slide 7) | **Plural masculine imperative** ("עקבו", "תגיבו") |

CTAs always plural — they address the audience as a community.

### Banned words in copy

| Banned | Use instead |
|---|---|
| ברנד | מותג |
| טמפלייט | תבנית / template (English) |
| קונטרול | שליטה |
| האק | טריק / שיטה |
| ETF-ים, IRA-ים, 401k-ים | English plural-s (ETFs, IRAs, 401ks) |
| בלי מאמץ | בקלות |
| ממש (as filler) | drop or restructure |
| הנה (as sentence opener) | drop |

### Banned structural patterns

- **Time-arrow proofs**: ❌ `12 שנים → פנסיה` — does not land. Use a single
  number with context.
- **Two-fragment summaries** when a flowing sentence fits.
- **Generic "in the world today" openers** — `בעולם של היום...`, `כאשר אנו
  מדברים על...`.

### Word-choice preferences (small things that mark native Hebrew)

| Preferred | Less good |
|---|---|
| ואלו הם | ואלה |
| משלושת הטעויות האלו | משלוש הטעויות |
| מכל ריצה | בכל ריצה |
| חובה | חיוני / קריטי |
| מגדירים פעם אחת | פעם אחת מגדירים |
| בקלות | בלי מאמץ |
| השקעה / השקעות | אינווסטמנט |

### Hebrew + English in the same line (BIDI)

When a Hebrew preposition meets an English finance term, **insert a space**
in headlines — never glue:
- ✅ `ל Roth IRA`
- ✅ `ב 401k`
- ✅ `עם ETF`
- ❌ `ל-Roth IRA` *(in image-rendered text — dash reads as visual noise)*

For inline mentions in dense paragraphs, the dash form is acceptable.

## Slide 1 — Hook patterns

### Layout family A — Three-layer (kicker + main + subhead)

The signature carousel-cover layout. Used when the hook compresses to a
punchy 3-part headline.

```
KICKER  (small, top, optional)
MAIN HEADLINE (massive, English brand term often)
SUBHEAD (italic, bottom, Hebrew payoff line)
```

#### Examples (Layout A)

**(1) Numbered roundup — gold-standard**
| | |
|---|---|
| Kicker | טופ 5 |
| Main | **Index Funds** |
| Subhead | *שמכפילים את החיסכון לפנסיה* |
| Pattern | `טופ N` + `[ENGLISH BRAND TERM]` + Hebrew benefit line |

**(2) Stat-shock**
| | |
|---|---|
| Main | **70%** |
| Subhead | *מהאמריקאים לא מקסמו את ה-401k שלהם* |
| Pattern | massive number + Hebrew implication |

**(3) Bold transformation**
| | |
|---|---|
| Main | **Roth IRA** |
| Subhead | *הוא לא מה שחשבת* |
| Pattern | brand name + radical reframing |

**(4) Audience-specific**
| | |
|---|---|
| Main | **Roth IRA** |
| Subhead | *לעצמאים* |
| Pattern | `[BRAND]` + `ל[AUDIENCE]` (one-word audience tag) |

**(5) Mistakes**
| | |
|---|---|
| Main | **3 טעויות** |
| Subhead | *שכולם עושים עם 401k* |
| Pattern | `N טעויות` + relative clause |

### Layout family B — Long narrative (tweet-style)

Single flowing sentence rendered across multiple lines, smaller font.

#### Examples (Layout B)

**(6) Personal transformation tutorial**
> `ככה הפכתי את התיק שלי בעזרת 3 ETFs בלבד`
- Pattern: `ככה [VERB] [SCOPE] בעזרת [BRAND/STRATEGY]`
- First-person discovery framing

**(7) Counter-intuitive personal**
> `הפסקתי לחסוך עוד ב-IRA. הסיבה: Mega Backdoor.`
- Pattern: `הפסקתי [OLD HABIT]. הסיבה: [NEW THING]`
- Two-clause structure with em-dash-style colon turn — strong hook

**(8) Long first-person achievement**
> `אני הרגע סגרתי את ה-401k המקסימום של 2025 בעזרת אוטומציה אחת`
- Pattern: `אני הרגע [VERB] [STAT/SCOPE] בעזרת [BRAND/STRATEGY]`
- Stat embedded mid-sentence, brand name lands at the end

## Slide 2 — Hook-proof / transition

Short numeric/quantitative headline + one-line transition. The headline must
read self-contained — without the hook context. Banned: time-arrows,
`כפי שאמרתי`, `ממשיכים`, `השלב הבא`.

| # | Headline | Transition |
|---|---|---|
| 1 | **47,000 משתמשים בודקים את זה כל חודש** | *ואלו הם 5 הדברים שהם רואים.* |
| 2 | **70% מהאמריקאים תקועים פה** | *וכמעט תמיד זה אחת משלושת הטעויות האלו.* |
| 3 | **פחות מ-3 דקות הגדרה** | *אבל אף אחד לא טורח לעשות את זה.* |
| 4 | **$5,000 בשנה, בקלות** | *3 ETFs. אוטומציה אחת. זה הכל.* |
| 5 | **Vanguard עצמם פתחו את זה לכולם** | *תוך 24 שעות זה הפך לברירת מחדל.* |
| 6 | **בדקתי 50 IRAs ורק אחד שווה את זה** | *וכמעט אף אחד לא מדבר עליו.* |

## Slide 3–7 — Tip examples

Tip pattern: **numbered headline + one-line body**. One concrete idea per
slide. Mostly Hebrew body; English only for unavoidable specific terms.

| # | Headline | Body | Pattern |
|---|---|---|---|
| 1 | **#1 — Roth IRA** | *פתח אחד היום. החיסכון לפנסיה ייגדל בלי שתשלם מס על הריבית.* | Account type + what-it-does. One crisp claim. |
| 2 | **Auto-deposit אחרי כל משכורת** | *מגדירים פעם אחת. אחר כך זה רץ לבד כל חודש.* | Mechanism description. Verb-first word order. |
| 3 | **#2 — לא מקסמים את ה-Match של 401k** | *תפקיד אחד שלך: להגיע לסכום שהמעסיק תואם. אחרת אתה משאיר כסף על השולחן.* | Mistake + fix in one line. Imperative singular masculine. |
| 4 | **#3 — Index funds** | *VTSAX מחזיק את כל השוק האמריקאי בקרן אחת. ETF VOO זול יותר. שניהם עובדים.* | Comparison tip — the place where English density is OK in body (specific fund tickers). |
| 5 | **#2 — אוטומציה לקרן חירום** | *שלוש משכורות בחיסכון, נגישות מיד. Vanguard, Wealthfront — כל אחד יודע לעשות את זה.* | Workflow tip — what + why. |
| 6 | **#5 — Mega Backdoor** | *במקום לחסוך 23,000 בלבד, מכניסים 70,000 לשנה לחשבון Roth. רק עובדים בחברות עם תוכנית.* | Comparison structure `במקום X, Y` — flowing explanatory body. |

### Tip body checklist

Before approving a tip body, verify:

- [ ] Body is **mostly Hebrew** (≤ 1–2 English finance terms unless headline is comparative)
- [ ] **One claim**, not a summary + supporting line
- [ ] Verb-first when describing process (`מגדירים`, `פותחים`, `מקסמים`)
- [ ] No banned words (`ברנד`, `טמפלייט`, `בלי מאמץ`, `ממש`)
- [ ] Could be illustrated with a concrete visual (screenshot, chart, account-opening flow)

## Slide 7 (or 8) — CTA examples

Two patterns work for this voice. **Always plural imperative**. CTAs do not
use exclamation points.

### Pattern A — Follow CTA

Minimal. Headline is the action. Subhead is the handle.

| Headline | Subhead |
|---|---|
| **עקבו לעוד.** | *@personalfinancetips* |

### Pattern B — Lead-magnet CTA

Fixed formula:

> `תגיבו "[KEYWORD]" ואני אשלח לכם את [ה-X / המדריך המלא]`

The keyword is **always Capitalized** (`Roth`, `Vanguard`, `ETF`, `401k`).
The deliverable is either `המדריך המלא` (default) or a specific named
artifact (`ה checklist`, `ה template`, `ה list`).

#### Examples

| Variant | Full text |
|---|---|
| Generic guide | **תגיבו "Roth" ואני אשלח לכם את המדריך המלא** |
| Specific guide | **תגיבו "ETF" ואני אשלח לכם את המדריך המלא** |
| Specific artifact | **תגיבו "401k" ואני אשלח את ה checklist** |

No subhead needed for lead-magnet — the action is the message. Handle
optional.

### Banned CTA patterns

These were tested and rejected:

- ❌ Save CTAs (`שמור את זה לעצמך`)
- ❌ Tag-a-friend CTAs (`תייג חבר שצריך לחסוך`)
- ❌ Open-ended question CTAs (`איזה ETF אתה מחזיק?`)
- ❌ Single-imperative CTAs (singular `עקוב`, `שלח`)

## How Pass B uses this file

1. **Read `brand-voice-he.md` first** — voice rules.
2. **Read this file** — concrete patterns.
3. **Read `english-terms-whitelist.json`** — which English finance terms
   stay English.
4. **For each slide brief**:
   - Identify role (hook / proof / tip / CTA) and the closest pattern shape.
   - Imitate the closest example structurally — same word order, density, rhythm.
   - Apply cross-cutting rules (English density, address, banned words, BIDI).
5. **Run the 3-second test** from `brand-voice-he.md` before output.

When the input content doesn't fit any pattern here, **say so explicitly**
in the output — don't invent a pattern silently.
